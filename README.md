# Italian Practice A1/A2

A Duolingo-style static practice tool for Italian A1/A2 exam preparation.

The lesson content is based on the provided `Italian_A1_A2_Exam_Vocabulary_and_Grammar.docx` pack, with units for pronunciation, vocabulary, articles, plurals, adjectives, verbs, prepositions, dialogues, passato prossimo, and mixed exam review.

## Run locally

Open `index.html` in a browser.

## GitHub Pages

This is a static site. To publish it on GitHub Pages, enable Pages for the repository and choose the `main` branch root as the source.

## Features

- Lesson path with unlock-style progress
- XP, hearts, streak, and completion tracking
- Mistake review bank stored in localStorage
- Multiple choice, typing, sentence builder, and listening-style prompts
- A1/A2 exam mode with mixed practice
